ansible-role-tzdata
===================

Timezone configuration.

Requirements
------------

None.

Role Variables
--------------

* `tzdata_timezone` is a valid timezone name

Dependencies
------------

None.

Example Playbook
----------------

    ---
    - hosts: localhost
      become: true
      roles:
        - role: pkorobeinikov.tzdata
          tzdata_timezone: America/Cancun
